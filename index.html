如何提高外语交流能力

  1. 如果有条件，多看电视，多听广播。
  2. 尽量多一个人出门办事，尽可能选择没有国人伴随的情况以强迫自己必须练习听说。
  3. 只要你愿意和有胆去跟别人交朋友，一年半载下来你的意大利语一定会比现在有很大的进步。
  4. 一个有效的学习循环是：讲不出 -> 听别人讲 -> 自己重复类似语句 -> 完成话题的掌握 -> 尝试新的话题。...
